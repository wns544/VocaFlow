# VocaFlow

iOS 기말 프로젝트로 제작한 영어 단어 학습 앱입니다.

카드형 학습과 직접 입력하는 학습을 같이 사용해서 단어를 반복해서 복습할 수 있도록 만들었습니다.
기본적인 화면 구성은 Storyboard / Interface Builder를 중심으로 작업했습니다.

## 1. 프로젝트 개요

영어 단어를 외울 때 단순히 목록만 보는 방식은 금방 지루해질 수 있다고 생각했습니다.
그래서 짧게 넘기는 카드 학습과, 직접 뜻을 입력해 보는 타이핑 학습을 같이 넣었습니다.

단어장을 한 번에 전부 공부하는 것이 아니라 세션 단위로 나누어 학습할 수 있게 했고,
모르는 단어는 다시 학습 대기열로 들어가도록 구성했습니다.

## 2. 주요 기능

- 쇼츠 학습
  - 단어 카드를 넘기듯이 보면서 학습합니다.
  - 위로 스와이프하면 외운 단어로 처리합니다.
  - 아래로 스와이프하면 모르는 단어로 처리하고, 다시 나중에 나오도록 했습니다.
  - 예문 먼저 보기 설정을 켜면 카드 앞면에서 예문을 먼저 볼 수 있습니다.

- 타이핑 학습
  - 단어와 예문을 보고 뜻을 직접 입력합니다.
  - 오답이거나 모르겠음을 선택하면 해당 단어는 다시 뒤쪽 순서로 들어갑니다.
  - 정답 확인 후에는 정답, 발음, 예문, 예문 해석을 확인할 수 있습니다.

- 단어장 관리
  - CSV 파일을 이용해서 새 단어장을 추가할 수 있습니다.
  - 단어장별로 세션을 나누어 학습할 수 있습니다.
  - 사용자가 추가한 단어장은 이름 변경, 삭제, 순서 변경이 가능합니다.
  - 기본 단어장은 실수로 지우지 않도록 삭제할 수 없게 했습니다.

- 학습 보조 기능
  - D-day 이름과 날짜를 설정할 수 있습니다.
  - 연속 학습일을 표시합니다.
  - 학습 기록을 초기화할 수 있습니다.
  - 쇼츠 학습에서 사용할 빠른 학습 단어장을 따로 지정할 수 있습니다.

## 3. 화면 구성

- VocaFlow 탭
  - 앱의 첫 화면입니다.
  - 쇼츠 학습과 타이핑 학습을 바로 시작할 수 있습니다.
  - 설정한 D-day와 연속 학습일을 확인할 수 있습니다.

- 단어장 탭
  - 기본 단어장과 사용자가 추가한 단어장을 확인합니다.
  - 단어장을 누르면 먼저 세션 목록이 보이도록 구성했습니다.
  - 단어목록 버튼을 통해 단어 전체 목록도 확인할 수 있습니다.
  - CSV 파일을 추가해서 새 단어장을 만들 수 있습니다.

- 설정 탭
  - 세션 단어 수를 조절할 수 있습니다.
  - D-day 사용 여부, 이름, 날짜를 설정할 수 있습니다.
  - CSV 삽입 형식을 확인할 수 있습니다.
  - 학습 기록 초기화와 앱 정보를 확인할 수 있습니다.

## 4. CSV 단어장 형식

CSV 파일은 다음 순서로 작성합니다.

```text
단어 / 뜻 / 발음 / 예문 / 예문뜻
```

1열부터 3열까지인 단어, 뜻, 발음은 반드시 입력해야 합니다.
4열과 5열인 예문, 예문뜻은 없어도 추가할 수 있습니다.

예시는 다음과 같습니다.

```csv
apple,사과,apple,This is an apple.,이것은 사과입니다.
study,공부하다,study,I study English every day.,나는 매일 영어를 공부합니다.
```

## 5. 데이터 출처

기본 단어장은 GitHub 공개 저장소 `first20hours/google-10000-english`의 빈출 영어 단어 목록을 참고했습니다.

단어의 뜻, 발음, 예문, 예문 해석은 앱의 학습 화면에 맞게 따로 정리했습니다.

## 7. 구현하면서 신경 쓴 부분

- 학습 중에는 탭바가 보이지 않도록 해서 학습 화면에 집중하게 했습니다.
- 모르는 단어는 바로 다음에 나오지 않고, 남은 단어 사이에 다시 섞이도록 했습니다.
- 단어장 세션을 나누어 많은 단어도 조금씩 학습할 수 있게 했습니다.
- 설정 화면은 학습 설정, D-day, 단어장, 기록, 앱 정보 영역으로 나누어 정리했습니다.
