## 개발 의도

VocaFlow는 지루하게 느껴지는 영어 단어 암기를 더 가볍고 익숙한 방식으로 바꾸기 위해 기획한 앱입니다.

기존의 단어 학습은 긴 단어 목록을 반복해서 보는 방식이 많아, 학습을 시작하기 전부터 부담이 생기기 쉽습니다. 그래서 현재 많은 사람들이 익숙하게 사용하는 쇼츠나 릴스의 짧고 빠른 넘김 경험에서 아이디어를 얻었습니다.

사용자는 단어 카드를 하나씩 확인하고, 위아래 스와이프만으로 외운 단어와 다시 볼 단어를 구분할 수 있습니다. 핵심은 긴 시간 공부하는 것이 아니라, 짧게 시작하고 반복할 수 있는 부담 없는 학습 루틴을 만드는 것입니다.

## 프로젝트 개요

VocaFlow는 영어 단어를 카드 학습과 직접 입력 퀴즈로 반복 학습할 수 있도록 만든 iOS 앱입니다.

앱의 핵심 문장은 다음과 같습니다.

> 외운 단어는 넘기고, 모르는 단어는 다시 만나게 한다.

단어장을 한 번에 모두 외우는 방식이 아니라 세션 단위로 나누어 학습하며, 모르는 단어는 다시 학습 큐에 들어가도록 구성했습니다. 이를 통해 사용자는 짧은 시간에도 부담 없이 학습을 시작하고, 헷갈리는 단어를 자연스럽게 반복 복습할 수 있습니다.

## 주요 기능

### 1. 카드 기반 스와이프 학습

- 단어 카드가 한 장씩 표시됩니다.
- 카드를 터치하면 뜻, 발음, 예문, 예문 해석을 확인할 수 있습니다.
- 위로 스와이프하면 암기 완료 단어로 처리됩니다.
- 아래로 스와이프하면 다시 보기 단어로 처리됩니다.
- 다시 보기 단어는 학습 큐에 다시 삽입되어 같은 세션 안에서 다시 등장합니다.
- 마지막 카드 동작을 되돌릴 수 있습니다.

### 2. 직접 입력형 타이핑 퀴즈

- 단어와 예문을 보고 뜻을 직접 입력합니다.
- 정답이면 암기 완료 단어로 처리됩니다.
- 오답이거나 모르겠음을 선택하면 다시 보기 단어로 처리됩니다.
- 틀린 단어는 문제 큐에 다시 삽입되어 반복 출제됩니다.
- 퀴즈가 끝나면 학습 결과를 확인할 수 있습니다.

### 3. 단어장 관리

- 기본 단어장을 제공합니다.
- CSV 파일을 불러와 사용자 단어장을 추가할 수 있습니다.
- 단어장 이름 변경, 삭제, 순서 변경이 가능합니다.
- 빠른 학습에 사용할 단어장을 지정할 수 있습니다.
- 단어장별로 세션 완료 현황을 확인할 수 있습니다.

### 4. 세션 단위 학습

- 긴 단어장을 일정한 단어 수의 세션으로 나누어 학습합니다.
- 세션당 단어 수는 설정 화면에서 조절할 수 있습니다.
- 완료한 세션은 기록으로 저장됩니다.
- 홈 화면에서 다음에 학습할 세션을 바로 시작할 수 있습니다.

### 5. 학습 보조 기능

- D-day 이름과 날짜를 설정할 수 있습니다.
- 연속 학습일을 확인할 수 있습니다.
- 단어 검색과 상태별 필터링을 지원합니다.
- 전체 학습 기록을 초기화할 수 있습니다.

## 화면 구성

### 홈 화면

- VocaFlow의 시작 화면입니다.
- 빠른 학습 단어장과 다음 세션을 보여줍니다.
- 카드 학습과 타이핑 퀴즈를 바로 시작할 수 있습니다.
- D-day와 연속 학습일을 확인할 수 있습니다.

### 카드 학습 화면

- 단어 카드 중심의 학습 화면입니다.
- 카드 터치로 정답 정보를 확인합니다.
- 위로 스와이프하면 암기 완료, 아래로 스와이프하면 다시 보기로 처리됩니다.
- 진행률과 학습 상태를 함께 보여줍니다.

### 타이핑 퀴즈 화면

- 단어의 뜻을 직접 입력하는 복습 화면입니다.
- 정답, 오답, 모르겠음 결과에 따라 단어 상태가 변경됩니다.
- 오답 단어는 다시 출제되어 반복 학습이 이루어집니다.

### 결과 화면

- 전체 학습 단어 수를 보여줍니다.
- 암기한 단어 수를 보여줍니다.
- 다시 볼 단어 수를 보여줍니다.
- 암기 비율을 보여줍니다.

### 단어장 화면

- 기본 단어장과 사용자 추가 단어장을 확인합니다.
- 단어장별 세션 목록과 단어 목록을 볼 수 있습니다.
- 커스텀 단어장은 수정 및 삭제할 수 있습니다.

### 설정 화면

- 세션당 단어 수를 설정합니다.
- D-day 사용 여부, 이름, 목표 날짜를 설정합니다.
- CSV 작성 형식을 확인합니다.
- 학습 기록을 초기화합니다.

## CSV 단어장 형식

CSV 파일은 다음 순서로 작성합니다.

```csv
term,meaning,reading,example,exampleMeaning
apple,사과,apple,This is an apple.,이것은 사과입니다.
study,공부하다,study,I study English every day.,나는 매일 영어를 공부합니다.
```

필수 항목:

- term: 단어
- meaning: 뜻
- reading: 발음

선택 항목:

- example: 예문
- exampleMeaning: 예문 해석

CSV 파일을 이용하면 시험 단어장, 전공 용어, 스포츠 팀/선수 이름처럼 사용자가 원하는 주제의 단어장을 직접 만들 수 있습니다.

## 기대 효과

VocaFlow는 영어 단어 학습을 더 가볍고 직관적인 경험으로 바꾸는 것을 목표로 합니다.

사용자는 앱을 켜고 바로 짧은 세션을 시작할 수 있으며, 외운 단어와 모르는 단어를 간단한 스와이프 동작으로 구분할 수 있습니다. 또한 모르는 단어가 자연스럽게 다시 등장하기 때문에 반복 학습 흐름을 유지할 수 있습니다.

이를 통해 단어 암기를 단순히 지루한 반복 작업이 아니라, 짧게 시작하고 계속 이어갈 수 있는 학습 루틴으로 만들고자 했습니다.
